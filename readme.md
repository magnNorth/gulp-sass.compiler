# Basic style merger

This basic style merger works to compile and compress '*.SCSS' to a sytles.css file. 
This is done with gulp and gulp sass and gulp concat. 

## Setup 

```
git clone $this...

cd main-sass

npm install 
```


## Use

one off compile.

```
npm run build 
```

Watch 

```
npm run watch
```


## Deploy

THe build drops the new file in to deploy folder.
